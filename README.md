# Python物业管理系统 python47

## 项目概述

本项目是一个基于Python Flask框架的物业管理系统，致力于帮助物业管理部门高效地完成日常工作，提升管理质量和效率。

## 技术栈

- **前端**：Vue.js + Element UI + Axios
- **后端**：Python Flask + Flask-Security + Flask-SQLAlchemy
- **数据库**：MySQL
- **环境依赖**：Python 3.12，Node.js 12
- **开发工具推荐**：VSCode / PyCharm

## 功能模块

### 系统管理

- 用户登录与权限控制
- 系统设置与配置

### 房屋与车位管理

- 房源信息录入与修改
- 车位分配与信息管理

### 费用管理

- 费用类型设置
- 费用收取与查询

### 业主信息管理

- 业主信息录入与更新
- 信息查询与统计

### 反馈与维修管理

- 业主反馈收集
- 维修工单生成与跟踪

### 公告管理

- 公告发布与编辑
- 公告查阅记录

## 系统角色

系统主要包含以下角色：
- 系统管理员
- 物业工作人员
- 业主

各角色拥有不同的权限和功能模块。
